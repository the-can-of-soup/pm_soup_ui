# SoupUI
A user interface "library" for [PenguinMod](https://penguinmod.com/).

## Installation
1. Go to my project hosted here *(The project isn't public yet)*, or download [SoupUI.pmp](https://github.com/the-can-of-soup/pm_soup_ui/raw/refs/heads/main/SoupUI.pmp) and open it in the [editor](https://studio.penguinmod.com/editor.html).
2. Save the sprite "SoupUI" to your backpack.
3. Go to your project you would like to install SoupUI into.
4. Add the following extensions to your project (if they're not already added):
   * My Blocks+
   * Objects
   * Arrays
   * Lambda
   * Temporary Variables (the blue one)
   * Labels
   * Operators Expansion
   * Dropdown Maker (in SP's extension collection)
   * Targets
   * Vector
   * Color
5. Import the sprite from your backpack.
